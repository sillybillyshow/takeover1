# Privacy Policy **Last Updated:** March 2026 ## 1. Introduction This Privacy Policy describes how this web application ("the App") handles information. The App displays publicly available TikTok follower count data for a single TikTok account. It does not require user accounts, logins, or personal information from visitors. ## 2. Information We Collect The App does **not** collect, store, or process any personal data from users. Specifically: * We do not collect names * We do not collect email addresses * We do not collect TikTok account information from visitors * We do not use cookies for tracking * We do not use analytics software ## 3. TikTok Data The App uses the official TikTok API to retrieve follower count data for a single authorized TikTok account. The data retrieved: * Follower count * Public account statistics No private TikTok user data is accessed or stored. ## 4. Data Storage The App does not maintain a database of user information. Follower count data may be temporarily cached for performance purposes but is not stored long-term. ## 5. Third-Party Services The App is hosted using: * GitHub Pages * Cloudflare Workers These platforms may process technical data such as IP addresses as part of standard web hosting operations. Please refer to their respective privacy policies for details. ## 6. Children's Privacy This App is not directed at children under 13 and does not knowingly collect personal data from children. ## 7. Changes to This Policy This Privacy Policy may be updated periodically. Any changes will be reflected on this page with an updated revision date. ## 8. Contact For questions regarding this Privacy Policy, please contact the developer via the GitHub repository associated with this App.
